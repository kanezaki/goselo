GOSELO Simulator

This demo for our paper GOSELO uses the WebDNN library, which fully runs on your web browser.
When you start the demo program, an agent (red circle) moves from a starting point to a goal (blue circle).

Get Started

  1. Wait until the library is loaded. (This may take several minutes.)
  2. Click "MAP" button and select a map image.
  3. Click "INIT" button.
  4. Click "PLAY" button to move the agent toward the goal.
  5. Click "STOP" button to stop the agent.
